# catclassifier
This classifier is based on Andrew Ng Deep Learning course Assignment<br/>
It is intent to identify cat images and non-cat images.<br/><br/>
I use 2 - Layer Neural Network with 7 nodes in hidden layer.
<br/>
Learning rate = 0.075
